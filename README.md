# Code Manager SaaS

Aplicação SaaS completa para gerenciamento de códigos.

## Stack
- Node.js
- Express
- Better-SQLite3
- EJS
- HTML, CSS, JS
- Express-session
- bcrypt
- Helmet
- Compression
- Express-rate-limit
- Multer
- Dotenv

## Deploy
Compatível com Render.  
Utiliza `process.env.PORT`.

## Funcionalidades
- Autenticação segura
- Dashboard com estatísticas
- Geração e importação de códigos
- Exportação em TXT
- Lotes e histórico
- API REST
